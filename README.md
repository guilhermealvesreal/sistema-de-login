# Sistema de Login e Cadastro

Sistema de login e cadastro desenvolvido com HTML, CSS e JavaScript, utilizando o localStorage para simular um banco de dados e controle de autenticação no front-end.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- localStorage

---

## Funcionalidades
- Cadastro de usuários
- Login com validação de email e senha
- Verificação de usuário já cadastrado
- Controle de sessão (usuário logado)
- Logout
- Exibição de dashboard após login
- Persistência de dados no navegador

---

## O que eu pratiquei neste projeto
- Manipulação do DOM
- Eventos e formulários
- Validação de dados
- Lógica de autenticação
- Uso do localStorage
- Controle de estados da aplicação
- Estruturação de código front-end
